# 商品管理模块

## 1. 模块概述

商品管理模块是 ERP 系统的核心基础模块，负责管理商品相关的基础数据，包括商品分类、品牌、产品信息。主要服务对象为仓库管理人员、采购人员和系统管理员。

---

## 2. 页面结构

### 2.1 页签定义

| 页签名称 | 用途说明 |
|---------|---------|
| 分类管理 | 管理商品分类的树形结构，支持多级分类、税务编码设置 |
| 品牌管理 | 管理商品品牌信息，包含Logo、描述、默认运费等 |
| 产品管理 | 管理产品基本信息，关联分类和品牌，包含规格管理 |

- 点击产品编号旁的三角图标可收起/展开规格列表

#### 2.4.6 新建/编辑产品表单

表单分为两个区域：**产品信息**、**规格信息**

##### 2.4.6.1 产品信息

| 字段名 | 类型 | 必填 | 默认值 | 说明 |
|-------|------|------|-------|------|
| 产品编号 | 文本输入 | 是 | 手动输入 | 产品编号（非唯一） |
| 产品名称 | 文本输入 | 是 | 空 | 产品名称 |
| 品牌 | 下拉选择+快速新增 | 是 | 空 | 选择所属品牌，支持快速新增品牌并同步到品牌管理。选择品牌后自动显示该品牌的Logo |
| 分类 | 下拉选择 | 是 | 空 | 选择所属分类。选择分类后自动填充该分类的税务编码 |
| 税务编码 | 文本输入（只读） | 否 | 自动填充 | 根据选择的分类自动填充税务编码 |
| 产品图片 | 文件上传 | 否 | 空 | 产品图片（选择品牌后自动显示品牌Logo） |

**说明：**
- 新建产品时：不显示产品状态字段，新建产品默认为"在售"状态
- 编辑产品时：显示产品状态字段，可修改产品状态（在售/作废）
- 选择品牌后，产品图片区域会自动显示该品牌的Logo
- 选择分类后，税务编码会自动填充

**唯一性规则说明：**
- 产品编号、规格编号、品牌三者联合唯一
- 当三者完全一致时，视为同一产品，执行覆盖更新操作
- 当三者任意字段不一致时，视为新建产品

##### 2.4.6.2 规格信息（子表格，可动态添加行）

| 字段名 | 类型 | 必填 | 默认值 | 说明 |
|-------|------|------|-------|------|
| 规格编号 | 文本输入 | 是 | 自动生成 | 格式：产品编号-包装 |
| 包装 | 文本输入 | 是 | 空 | 包装说明 |
| 包装单位 | 文本输入 | 是 | 空 | 包装单位 |
| 目录价格 | 金额输入 | 是 | 0.00 | 目录价格 |
| 操作 | 按钮 | - | - | 删除该行规格 |

**自动带入规则：**
- 填写完产品信息后，规格编号自动带入产品编号信息
- 填写完包装字段后，规格编号自动更新为"产品编号-包装"格式

#### 2.5.4 导入逻辑

| 逻辑项 | 说明 |
|-------|------|
| 校验规则 | 校验必填字段、品牌和分类是否存在、价格格式等 |
| 品牌Logo显示 | 根据品牌名称自动显示对应的Logo图标 |
| 税务编码填充 | 根据产品分类自动填充对应的税务编码 |
| 错误提示 | 显示错误行号和详细错误信息，支持修正后重新上传 |
| 数据保留 | 导入时保留新增数据和错误数据，失败项提供详细说明 |
| 导入结果 | 显示处理条数、新增成功条数、新增失败条数 |
| 唯一性规则 | 产品编号、规格编号、品牌三者联合唯一，三者完全一致时视为同一产品 |
| 重复处理 | 三者一致时覆盖原有产品更新信息，三者任意字段不一致则新建产品 |

## 3. 业务逻辑

### 3.1 数据关系

```
分类 (category)
    └── 产品 (product)
            └── 规格 (spec)
品牌 (brand)
    └── 产品 (product)
```

### 3.2 状态定义

#### 3.2.1 分类状态

| 状态 | 含义 | 说明 |
|------|------|------|
| 启用 | 正常使用 | 可关联产品 |
| 停用 | 停止使用 | 不可关联新产品 |

#### 3.2.2 品牌状态

| 状态 | 含义 | 说明 |
|------|------|------|
| 启用 | 正常使用 | 可关联产品 |
| 停用 | 停止使用 | 不可关联新产品 |

#### 3.2.3 产品状态

| 状态 | 含义 | 说明 |
|------|------|------|
| 在售 | 正常销售 | 可下单购买 |
| 下架 | 停止销售 | 不可下单 |
| 作废 | 已作废 | 不可使用，仅用于历史查询 |

### 3.3 权限控制

| 功能模块 | 普通用户 | 管理员 |
|---------|---------|--------|
| 查看分类/品牌/产品 | ✓ | ✓ |
| 新建/编辑分类/品牌/产品 | ✓ | ✓ |
| 删除分类/品牌/产品 | ✗ | ✓ |
| 批量上传商品 | ✗ | ✓ |
| 复制产品 | ✗ | ✓ |
| 作废产品 | ✗ | ✓ |

## 4. 数据模型

### 4.1 实体：商品分类（category）

| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| id | bigint | 是 | 主键 |
| name | varchar(100) | 是 | 分类名称 |
| parent_id | bigint | 否 | 父分类ID，一级分类为0 |
| level | tinyint | 是 | 层级：1-一级，2-二级，3-三级 |
| tax_code | varchar(50) | 否 | 税务编码 |
| sort_order | int | 否 | 排序序号 |
| display | tinyint | 是 | 是否展示：0-否，1-是 |
| status | tinyint | 是 | 状态：0-停用，1-启用 |
| created_at | datetime | 是 | 创建时间 |
| updated_at | datetime | 是 | 更新时间 |

### 4.2 实体：品牌（brand）

| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| id | bigint | 是 | 主键 |
| name | varchar(100) | 是 | 品牌名称 |
| logo | varchar(500) | 否 | Logo文件路径 |
| description | varchar(500) | 否 | 品牌描述 |
| purchaser_id | bigint | 否 | 采购人员ID |
| purchaser_name | varchar(50) | 否 | 采购人员名称（冗余） |
| default_freight | decimal(10,2) | 否 | 默认运费 |
| status | tinyint | 是 | 状态：0-停用，1-启用 |
| product_count | int | 否 | 商品数量（统计字段） |
| created_at | datetime | 是 | 创建时间 |
| updated_at | datetime | 是 | 更新时间 |

### 4.3 实体：产品（product）

| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| id | bigint | 是 | 主键 |
| code | varchar(50) | 是 | 产品编号（非唯一） |
| name | varchar(200) | 是 | 产品名称 |
| brand_id | bigint | 是 | 品牌ID |
| brand_name | varchar(100) | 是 | 品牌名称（冗余） |
| category_id | bigint | 是 | 分类ID |
| category_name | varchar(100) | 是 | 分类名称（冗余） |

| created_at | datetime | 是 | 创建时间 |
| updated_at | datetime | 是 | 更新时间 |

**唯一约束：** product.code + spec.code + product.brand_id 联合唯一

### 4.4 实体：规格（spec）

| 字段名 | 类型 | 必填 | 说明 |
|-------|------|------|------|
| id | bigint | 是 | 主键 |
| code | varchar(100) | 是 | 规格编号，格式：产品编号-包装 |
| product_id | bigint | 是 | 产品ID |
| product_name | varchar(200) | 是 | 产品名称（冗余） |
| package_info | varchar(200) | 是 | 包装说明 |
| package_unit | varchar(20) | 是 | 包装单位 |
| catalog_price | decimal(12,2) | 是 | 目录价格 |
| status | tinyint | 是 | 状态：0-无效，1-有效 |
| created_at | datetime | 是 | 创建时间 |
| updated_at | datetime | 是 | 更新时间 |
